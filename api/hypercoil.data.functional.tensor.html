Tensor casting and binding#

hypercoil.data.functional.to_tensor(data, dtype=None, device=None, dim='auto')#

Change the input data into a tensor with the specified type and minimum dimension.

Parameters
dataobject

Object that can be cast to a Tensor. If this is a pandas DataFrame, only the values are cast to a Tensor.

dtypetorch datatype

Output tensor datatype.

dimint or ‘auto’ (default ‘auto’)

Minimum dimension of the output tensor. If this is ‘auto’, then no adjustments are made to the output tensor dimension. Otherwise, a singleton dimension is iteratively added until the tensor dimension is greater than or equal to the specified argument.

Returns
tensorTensor

PyTorch Tensor containing the input data.

hypercoil.data.functional.to_named_tensor(data, dtype=None, device=None, dim='auto', names=None)#

Change the input data into a named tensor with the specified type and minimum dimension.

Parameters
dataobject

Object that can be cast to a Tensor. If this is a pandas DataFrame, only the values are cast to a Tensor.

dtypetorch datatype

Output tensor datatype.

dimint or ‘auto’ (default ‘auto’)

Minimum dimension of the output tensor. If this is ‘auto’, then no adjustments are made to the output tensor dimension. Otherwise, a singleton dimension is iteratively added until the tensor dimension is greater than or equal to the specified argument.

nameslist(str) or None (default None)

List of names to assign the dimensions of the produced tensor.

Returns
tensorTensor

PyTorch Tensor containing the input data.

hypercoil.data.functional.consolidate_block(block)#

Consolidate a list block of tensors into a single tensor. If the tensors in the list block are of different sizes, smaller tensors are padded with NaN entries until all sizes are conformant before consolidation.

hypercoil.data.functional.extend_to_size(tensor, size)#

Extend a tensor in the positive direction until its size matches the specification. Any new entries created via extension are marked with NaN to denote that they were missing from the input; they can be populated by chaining this with a call to nanfill.

hypercoil.data.functional.extend_to_max_size(tensor_list)#

Extend all tensors in a list until their sizes are equal to the size of the largest tensor along each axis. Any new entries created via extension are marked with NaN to denote that they were missing from the input; they can be populated by chaining this with a call to nanfill.
